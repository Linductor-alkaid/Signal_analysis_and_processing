信号分析与处理 - 第三章：离散时间傅里叶变换(DTFT)

3.1 DTFT的定义

基本定义

序列 x[n] 的离散时间傅里叶变换(DTFT)定义为：

X(ejω) = ∑n=-∞∞ x[n]e-jωn

相应的逆变换(IDTFT)为：

x[n] = (1/2π) ∫-ππ X(ejω)ejωndω

其中：

  • ω：数字角频率（弧度），-π ≤ ω ≤ π
  • X(ejω)：x[n]的频谱函数
  • ejω：单位圆上的复指数

🔗 与z变换的关系

DTFT是z变换在单位圆上的特例：

X(ejω) = X(z)|z=ejω

当z变换的ROC包含单位圆时，DTFT存在。

3.2 物理意义

频域分析的本质

  1. 频谱分解：将离散信号分解为不同频率的复指数分量
  2. 频率响应：描述系统对不同频率分量的响应特性
  3. 能量分布：|X(ejω)|²表示信号在各频率上的能量分布
  4. 相位信息：∠X(ejω)表示各频率分量的相位关系

几何解释

DTFT可以理解为：

  • 将序列投影到单位圆上不同频率的复指数基函数
  • 每个频率ω对应单位圆上一个点 ejω
  • X(ejω)给出了该频率分量的复振幅

3.3 DTFT的重要特点

1. 周期性

X(ej(ω+2π)) = X(ejω)

原因：ej(ω+2π)n = ejωnej2πn = ejωn

因此，只需要分析一个周期内的频谱即可

2. 实序列的共轭对称性

当 x[n] 为实序列时：

X(ej(-ω)) = X*(ejω)

这意味着：

  • 幅度谱：|X(ejω)|为偶函数
  • 相位谱：∠X(ejω)为奇函数

3. 连续性

DTFT是连续函数，不同于离散傅里叶变换(DFT)的离散特性

DTFT特性可视化

幅度谱 |X(ejω)|

相位谱 ∠X(ejω)

3.4 存在条件

绝对可积条件

DTFT存在的充分条件是序列绝对可积：

∑n=-∞∞ |x[n]| < ∞

这保证了DTFT的存在性和一致收敛性。

⚠️ 特殊情况

某些不满足绝对可积条件的序列，其DTFT可能以广义函数（冲激函数）的形式存在：

  • 常数序列：x[n] = 1
  • 正弦序列：x[n] = cos(ω₀n)
  • 阶跃序列：x[n] = u[n]

3.6 DTFT计算方法

计算步骤

  1. 检查存在条件：验证绝对可积性
  2. 应用定义：直接计算级数或查表
  3. 利用性质：时移、频移、对称性等
  4. 几何级数求和：对于指数类序列
  5. 分段处理：有限长序列分段计算

计算实例：矩形窗序列

问题：求矩形窗序列的DTFT

x[n] = {1, 0 ≤ n ≤ N-1; 0, 其他}

解：

X(ejω) = ∑n=0N-1 e-jωn = (1-e-jωN)/(1-e-jω)

化简得到：

X(ejω) = e-jω(N-1)/2 · sin(ωN/2)/sin(ω/2)

练习题目

练习1：DTFT基本计算

求下列序列的DTFT：

  1. x₁[n] = (0.5)nu[n]
  2. x₂[n] = (0.5)|n|
  3. x₃[n] = {1, -2≤n≤2; 0, 其他}
答案：
1. X₁(e^jω) = 1/(1-0.5e^(-jω))
2. X₂(e^jω) = (1-0.25)/(1-0.5e^(-jω)) + (1-0.25)/(1-0.5e^(jω)) - 1 = (1.75-cos(ω))/(1.25-cos(ω))
3. X₃(e^jω) = 1 + 2cos(ω) + 2cos(2ω)

练习2：存在性判断

判断下列序列的DTFT存在性：

  1. x₁[n] = n·(0.8)nu[n]
  2. x₂[n] = (1.2)nu[n]
  3. x₃[n] = (0.9)|n|
  4. x₄[n] = cos(πn/3)u[n]
答案：
1. 存在（|0.8| < 1，绝对可积）
2. 不存在（|1.2| > 1，不绝对可积）
3. 存在（双边指数序列，|0.9| < 1）
4. 不存在（周期序列，不绝对可积，但可能广义存在）

练习3：频谱性质应用

已知 x[n] = (0.5)nu[n] 的DTFT为 X(ejω) = 1/(1-0.5e-jω)

求下列序列的DTFT：

  1. y₁[n] = x[n-3]
  2. y₂[n] = ejπn/4x[n]
  3. y₃[n] = x[-n]
答案：
1. Y₁(e^jω) = e^(-j3ω)/(1-0.5e^(-jω))（时移性质）
2. Y₂(e^jω) = 1/(1-0.5e^(-j(ω-π/4)))（频移性质）
3. Y₃(e^jω) = 1/(1-0.5e^(jω))（时间反折性质）

练习4：频谱分析

矩形窗序列 x[n] = {1, 0≤n≤7; 0, 其他} 的幅度谱在哪些频率处为零？

答案：
幅度谱 |X(e^jω)| = |sin(4ω)/sin(ω/2)|
零点出现在：ω = 2πk/8, k = ±1, ±2, ±3, ...(除了k=0,±8,±16,...)
即：ω = π/4, π/2, 3π/4, π, 5π/4, 3π/2, 7π/4

3.7 学习要点总结

🎯 核心要点

  1. 定义理解：DTFT = z变换在单位圆上
  2. 周期性：频谱以2π为周期
  3. 对称性：实序列的幅度谱偶对称，相位谱奇对称
  4. 存在条件：绝对可积是充分条件
  5. 物理意义：频率域分析的基础工具
  6. 应用价值：系统频率响应分析的基础
