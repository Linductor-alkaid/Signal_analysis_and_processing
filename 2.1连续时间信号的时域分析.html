📊 连续时间信号的时域分析

1. 基本的连续时间信号

1.1 单位阶跃信号 u(t)

u(t) = {1, t ≥ 0; 0, t < 0}

1.2 单位冲激信号 δ(t)

δ(t) = {∞, t = 0; 0, t ≠ 0}
∫_{-∞}^{∞} δ(t)dt = 1

1.3 指数信号

x(t) = Ae^{at}, 其中A为幅度，a为指数参数
1 0.5

1.4 正弦信号

x(t) = A sin(ωt + φ), 其中A为幅度，ω为角频率，φ为初相位
1 2 0

重点记忆：

  • 单位阶跃信号是最基本的信号，用于描述开关特性
  • 单位冲激信号具有筛选特性：∫f(t)δ(t-t₀)dt = f(t₀)
  • 指数信号：a>0时递增，a<0时递减，a=0时为常数
  • 正弦信号的三要素：幅度、频率、相位

3. 连续时间信号的分解

3.1 奇偶分解

x(t) = x_e(t) + x_o(t)
偶分量：x_e(t) = [x(t) + x(-t)]/2
奇分量：x_o(t) = [x(t) - x(-t)]/2

奇偶分解性质：

  • 偶函数：f(-t) = f(t)，关于y轴对称
  • 奇函数：f(-t) = -f(t)，关于原点对称
  • 任何信号都可以唯一分解为奇分量和偶分量之和

4. 时域分析法——卷积

4.1 卷积定义

y(t) = x(t) * h(t) = ∫_{-∞}^{∞} x(τ)h(t-τ)dτ

4.2 卷积的图解法步骤

卷积图解法：

  • 1. 将h(τ)翻转得到h(-τ)
  • 2. 将h(-τ)时移t得到h(t-τ)
  • 3. 计算x(τ)与h(t-τ)的乘积
  • 4. 对乘积在整个τ轴上积分

4.3 卷积性质

重要性质：

  • 交换律：x(t) * h(t) = h(t) * x(t)
  • 结合律：[x(t) * h₁(t)] * h₂(t) = x(t) * [h₁(t) * h₂(t)]
  • 分配律：x(t) * [h₁(t) + h₂(t)] = x(t) * h₁(t) + x(t) * h₂(t)
  • 冲激函数性质：x(t) * δ(t) = x(t)
